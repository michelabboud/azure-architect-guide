## AWS to Azure AI Comparison

| AWS Service | Azure Equivalent | Migration Notes |
|-------------|-----------------|-----------------|
| Bedrock | Azure AI Foundry | Similar model catalog approach |
| SageMaker | Azure ML | Different SDK, similar concepts |
| Amazon Q | Microsoft Copilot | M365 integration unique to Azure |
| Lex | Copilot Studio | Low-code approach similar |
| Comprehend | Azure AI Language | Similar NLP capabilities |
| Rekognition | Azure AI Vision | Similar computer vision |
| Textract | Document Intelligence | Similar document processing |
