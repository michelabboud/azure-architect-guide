## AWS to Azure Certification Mapping

### Equivalent Certifications
| AWS | Azure | Notes |
|-----|-------|-------|
| SAA-C03 (SA Associate) | AZ-104 + AZ-305 | Need both for equivalent |
| SAP-C02 (SA Professional) | AZ-305 | Closest match |
| DVA-C02 (Developer) | AZ-204 | Developer focus |
| SOA-C02 (SysOps) | AZ-104 | Admin focus |
| DOP-C02 (DevOps) | AZ-400 | DevOps focus |
| SCS-C02 (Security) | AZ-500 | Security focus |
| ANS-C01 (Networking) | AZ-700 | Networking focus |
| DBS-C01 (Database) | DP-300 | Database focus |

## Study Schedules

### Aggressive (AWS Architect Background)
```
Week 1-2:  AZ-104 study
Week 3:    AZ-104 exam
Week 4-7:  AZ-305 study
Week 8:    AZ-305 exam
```

### Moderate
```
Month 1:   AZ-104 study
Month 2:   AZ-104 exam + start AZ-305
Month 3-4: AZ-305 study
Month 5:   AZ-305 exam
```

### Comprehensive
```
Month 1-2:   AZ-104
Month 3-4:   AZ-305
Month 5-6:   AZ-500 or AZ-700
```
